Netflix Data Analysis

Project Overview

This project analyzes the Netflix dataset to uncover insights about content trends, genres, countries, and growth patterns.
The goal is to understand how Netflix's content library has evolved over time.

---

Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib

---

Key Analysis Performed

* Data cleaning and handling missing values
* Analysis of Movies vs TV Shows
* Country-wise content distribution
* Year-wise release trends
* Genre analysis

---

Visualizations

* Bar chart: Movies vs TV Shows
* Top 10 countries producing content
* Release year distribution

---

 Key Insights

* Netflix’s catalog is dominated by movies compared to TV shows
* The United States contributes the highest amount of content
* Content production increased significantly after 2015
* Drama and international genres are highly prevalent
* A large portion of entries had missing director information

---

 Data Cleaning

* Handled missing values in columns like **director** and **country**
* Removed or filled incomplete records where necessary

---

 Dataset

Netflix dataset sourced from Kaggle
