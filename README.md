# Netflix Data Analysis

Project Overview

This project analyzes Netflix dataset to understand content trends, genres, and country distribution.

🛠 Tools Used

* Python
* Pandas
* NumPy
* Matplotlib

 Visualizations

* Movies vs TV Shows
* Top Countries
* Release Year Trends

 Key Insights

* Netflix’s catalog is dominated by movies
* USA is the leading content producer
* Content increased significantly after 2015
* Drama is one of the most common genres

 Dataset

Netflix dataset (Kaggle)
